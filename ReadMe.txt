Строка имеет форму
"Наимненование"/*пробел*/"условное количество целым числом"/*пробел*/"производитель"

Если не указать количество, то по умолчанию будет 1.
Если не указать производителя, то по умолчанию будет "---".
Можно указать производителя без кол-ва, но тогда надо ставить 2 пробела.
После производителя может быть сколько угодно пробелов, все будет производителем.
Если не соблюсти пробелы, считывание может быть непредсказуемым.
